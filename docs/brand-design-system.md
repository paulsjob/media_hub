# Brand Design System

Media Lab Lite uses the Professional Community Design System inspired by The Bench.

The interface should feel professional, modern, fast, polished, lightweight, and approachable. It should avoid heavy corporate coldness and avoid playful consumer-app styling.

## Color Foundations

- Primary accent: `#F37321`
- Light surface: `#f9f9ff`
- Soft blue neutral: `#cfdaf2`
- Deep navy and dark slate for typography and icons
- White cards on light neutral page backgrounds

Orange is reserved for primary calls to action, active states, key progress or success interactions, and important focus moments.

## Typography

The core UI typeface is Plus Jakarta Sans. It gives the product a modern, geometric, tech-forward, and highly legible feel.

## Layout

- Use an 8px spacing grid.
- Use 8px border radius for buttons, cards, inputs, previews, and panels.
- Prefer white cards on light neutral backgrounds.
- Use thin borders and low-elevation shadows.
- Avoid heavy gradients and oversized marketing hero language.

## Iconography

Icons should be thin-line, approximately 2px stroke, and used for clarity. They should support user action and scanning rather than decoration.
